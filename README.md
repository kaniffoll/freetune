# Free Tune

Музыкальный тюнер, основанный на Java библиотеке [Tarsos-DSP](https://github.com/JorenSix/TarsosDSP)

## Stack
- Kotlin
- Jetpack Compose
- Tarsos-DSP
- MVVM
- Dagger Hilt
